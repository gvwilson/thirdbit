How to Not Be Wrong About AI

This tutorial is a short introduction to research methods you can use to figure out whether AI-based programming tools are helping your team or not. Even if you don't do such studies yourself, we hope that understanding how they work will help you decide which claims about them to trust. Help is welcome, but please note that all contributors must abide by our Code of Conduct.
