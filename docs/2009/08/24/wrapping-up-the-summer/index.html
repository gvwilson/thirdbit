Wrapping Up the Summer

⇐ previous
Posted 2009-08-24
next ⇒

One by one and two by two our summer students are wrapping up:

  • Eben Hailemariam has moved the Eclipse feature diagram plugin to Google Code, and has posted a screencast showing the latest progress.
  • Maria Yancheva's MediaWiki search plugin has a screencast too.
  • Brent Mombourquette's Breadcrumbs plugin for Firefox is up on Google Code too.
  • Ainsley Lawson and Sarah Strong have finished their TracSNAP plugin (which draws social network graphs based on code), and uploaded it to TracHacks.

I’ll post more as they come in. Well done, everyone—it’s been a great four months.
