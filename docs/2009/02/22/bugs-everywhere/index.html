I’m still not convinced that distributed version control is the future, mostly because I don’t believe in the future: as always, there will be many mixed together, and since my focus is getting novice developers up the learning curve, I’m going to continue to rank initial ease of use over power in the hands of professionals. That said, Bugs Everywhere looks pretty cool: it’s a distributed bug tracking system that works with Arch, Bazaar, GIT, Mercurial, and RCS.
