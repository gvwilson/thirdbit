Comments

  1. The frameworks without grid support are (mostly) classless, i.e., authors are not expected to add classes to HTML elements to style their pages.
  2. Several frameworks set the width of a table based solely on its content without taking its caption into account.
  3. The Claude framework was generated by claude.ai. The only manual modifications are to use system fonts rather than a dozen chosen by Claude and to make the page wider.
